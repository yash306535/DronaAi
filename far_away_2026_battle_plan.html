P2
Sentinel Agent — behavioral fraud detection~3h
Detects tab-switching, copy-paste events, answer timing anomalies, and identical answer patterns across students using lightweight ML. Each flag has an explainability score (not just "cheating detected").
P2
Agent communication visualization~2h
Live log panel showing timestamped messages between agents (e.g. "Guardian → Herald: Face absent for 8s in Session #4"). This is the single most impressive visual for the demo — it proves multi-agent coordination is real.
P2
Analyst Agent — post-exam intelligence~2h
After exam ends, Analyst generates: per-student performance report, question difficulty heatmap, anomaly summary, and AI-written improvement suggestions per student. Shows scalability and practical value.
P2
Role-based access control~2h
Three roles: Admin (full dashboard), Invigilator (session monitoring only), Student (exam portal). JWT auth, different views per role. Shows engineering quality and production-readiness.
